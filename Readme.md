# BeFit

This app was built based on excercises as part of a "Advanced Programming" course at Merito WSB University.

## Database setup

This project uses sqlite by default. Please adjust the escaped filepath at appsettings.json "ConnectionStrings.DefaultConnection"

If you wish to use the provided connection string, create a writeable directory at "C:\dmz"
